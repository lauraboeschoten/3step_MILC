All code to run this example can be found in the file "MILC_3stap.R". 
The procedure starts with the datafile "1_bootstrap_samples.txt". 
This file contains an aggregated version of the original dataset and 5 
bootstrap samples taken by sampling from the original distribution.

The original (non-aggregated) dataset contains privacy sensitive information
so therefore this dataset can only be found at Statistics Netherlands and
is not published online. At Statistics Netherlands, we aggregated this dataset
and took bootstrap samples from it.
